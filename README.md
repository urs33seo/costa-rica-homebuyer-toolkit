# Costa Rica Homebuyer Toolkit (San José, Heredia & Puntarenas)

A practical set of checklists and templates to help you stay organized while exploring property in Costa Rica—especially in San José, Heredia, and Puntarenas.

## What’s inside
- Buyer due diligence checklist
- Property viewing checklist
- Document tracker template (CSV)
- Property comparison sheet (CSV)
- Closing-day checklist
- Quick glossary + questions to ask your agent

## How to use
Download the repo as a ZIP, or copy any checklist into your notes. Use the CSV templates in Google Sheets or Excel.
